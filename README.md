# BMI Calculator

A BMI Calculator project developed using Python.

## Features

### Beginner Version
- Command Line BMI Calculator
- BMI Calculation
- Category Detection
- Input Validation

### Advanced Version
- Tkinter GUI
- BMI History Storage
- CSV Data Storage
- BMI Trend Graph
- Multiple User Records

## BMI Formula

BMI = Weight (kg) / Height² (m)

## Technologies Used

- Python
- Tkinter
- CSV
- Matplotlib
- Git
- GitHub

## Run Beginner Version

python bmi_calculator.py

## Run Advanced Version

python bmi_gui.py
